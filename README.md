# Sales-Performance-Dashboard

## Overview
This project analyzes e-commerce sales performance using Python and Power BI.  
The dashboard provides insights into sales trends, customer behavior, and overall business performance through interactive visualizations and data analysis techniques.

---

## Objectives
- Analyze overall sales performance
- Identify top-performing categories and products
- Track revenue trends and business growth
- Create interactive dashboards for better decision-making

---

## Tools & Technologies
- Python
- Pandas
- Seaborn
- Power BI
- Google Colab

---

## Dataset
The project uses:

- `sales_dataset.csv`

The dataset contains e-commerce sales-related information used for analysis and dashboard creation.

---

## Features
- Interactive filters for better analysis
- Sales trend visualization
- Product/category performance analysis
- Data cleaning and preprocessing using Pandas
- Visual analytics using Seaborn
- Business insights through Power BI dashboards

---

## Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Dashboard Creation in Power BI
6. Business Insight Generation

## How to Run the Project

### Python Analysis
1. Open Jupyter Notebook/Google Colab
2. Install required libraries:

```bash
pip install pandas seaborn matplotlib
```

3. Run the notebook files

### Power BI Dashboard
1. Open the `.pbix` file in Power BI
2. Connect the dataset if required
3. Explore the interactive dashboard

## Key Insights
- Sales were highest in April (~26M) and then went down
- Sales are highest on Sundays and lowest on Thursdays 
- Cancellation highest was on Tuesday and Thursday was lowest
- Bengaluru (~6M) sells the most, followed by Hyderabad (~4M)
- Set and Kurta sell the most (~90K units)
- Amazon mostly uses Expedited delivery, Merchant prefer Standard delivery
- Almost all sales (~99%) come from B2C through Amazon
- ~75% Sales are Promoted
